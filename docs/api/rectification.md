# Rectification

Recovering what can be recovered from a birth time nobody wrote down.

Stellium rectifies **sect** — whether the chart is a day chart or a night chart —
and stops there. It does not attempt to recover a minute-level birth time, because
that is an ill-posed inverse problem: many times fit the evidence equally well, and
a method that returns one of them anyway is not solving the problem, it is hiding it.

Sect is different. It is a single bit, it moves the rulership of the whole chart, and
it can be argued for from evidence. See the [Rectification guide](../astrology/RECTIFICATION.md)
for what the technique claims and what it does not.
